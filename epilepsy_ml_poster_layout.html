Problem
50 million people worldwide have epilepsy. Seizures can cause falls, unconsciousness, and death if undetected. Automated EEG-based detection enables:

· Immediate nurse/family alerts
· Auto-stop driver assistance
· Emergency medication triggers

Core challenge: Only 20% of EEG windows contain seizures (1:4 imbalance). Signals are high-dimensional (178 measurements/sec) with redundant, noisy features.

Why undersampling wins despite removing 62% of data: Perfect 1:1 balance sharpens the LR decision boundary. Logistic regression is simple enough that clean balance outweighs lost information.

ROC-AUC barely changes across all strategies (83.1–83.4%) — the model's discriminative power is fixed. What changes is the precision-recall operating point. High recall is clinically essential — missing a seizure is far more dangerous than a false alarm.

Key Conclusions
① Preprocessing order matters: Pipeline A (ANOVA selection) beats Pipeline B (PCA) by 2.35 ROC-AUC points. Class-discriminative feature selection outperforms variance compression for classification tasks.
② Elastic Net is most robust: Ties L2 on F1 (57.1%), adds interpretable sparsity, most stable across correlated EEG features and different preprocessing choices.
③ Imbalance handling is critical: All balancing strategies boosted recall from 43% → 77–79%. Undersampling achieved best F1 (58.1%) — balance sharpens decision boundaries even at cost of data volume.
④ Linear model ceiling hit: Narrow performance range across all regularisation and pipeline conditions signals LR has reached its capacity limit. Next step: SVM (RBF kernel) or LSTM for temporal EEG patterns.
